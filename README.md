# 🍔 Swiggy Clone - Food Delivery UI

A modern, responsive **Food Delivery Web Application UI** inspired by Swiggy, built using **HTML, CSS, and Vanilla JavaScript**.

This project focuses on creating a clean and interactive user experience with features such as restaurant filtering, search, sorting, and cart functionality.

---

## 🚀 Features

### 🧭 Responsive Navigation Bar

* Sticky navigation bar.
* Brand logo.
* Delivery location section.
* Search functionality.
* User profile section.
* Dynamic cart counter.

### 🍕 Food Categories

* Horizontally scrollable category section.
* Filter restaurants by category.
* Active category highlighting.

### 🍽️ Restaurant Grid

* Fully responsive restaurant cards.
* Desktop: **4 cards per row**
* Tablet: **2 cards per row**
* Mobile: **1 card per row**

Each restaurant card includes:

* Restaurant image
* Restaurant name
* Rating
* Delivery time
* Cuisine types
* Price for two
* Add to Cart button

### 🔍 Search Functionality

* Search restaurants dynamically by name.
* Instant filtering without page reload.

### 📊 Sort Functionality

Users can sort restaurants by:

* Relevance (Default)
* Delivery Time
* Rating
* Cost: Low to High
* Cost: High to Low

### 🛒 Cart System

* Dynamic cart counter.
* Cart count increases whenever a user clicks **Add to Cart**.

### 📱 Fully Responsive

Optimized for:

* Desktop
* Tablet
* Mobile devices
